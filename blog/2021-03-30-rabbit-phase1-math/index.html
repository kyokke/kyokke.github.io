動画　2-8~11 : 特異値分解

  • 非正方行列に対する、固有値分解のようなもの

    • Mv=σu M v = \sigma uMv=σu

    • MTu=σv M^T u = \sigma vMTu=σv

    • このような単位ベクトル u,v u, v u,v を見つけられれば、

    • M=USVT M = U S V^T M=USVT

    • のように特異値分解できる。( U,V U, V U,V は直交行列 (ないしユニタリ行列))

  • 特異値の求め方の概要

    • MV=US MV = US MV=US , MTU=VST M^T U = V S ^TMTU=VST

    • 直交行列 の逆行列はその転置なので、

    • M=USVT M = USV^TM=USVT , MT=VSTUT M^T = VS^T U^TMT=VSTUT

    • これらの積を取ると,,

    • MMT=USSTUT MM^T = USS^T U^T MMT=USSTUT

    • -> MMT MM^T MMT を固有値分解すれば、よい。

  • 特異値分解の利用例

    • 例1 : 画像圧縮

      • 960x720の画像を行列とみて特異値分解

      • 大きい順に並べた特異値や、特異値のベクトルをを最初からk番目までつかう. (k+1 ~ はすてる)

      • 結果

        • 大きい方から 128 だけ取り出してもそれなりの画像に見える

        • 64でも、32でも、それなりにみえる。

        • 16まで落とすと, ぼやけてくる

    • 例2: 行列の類似性チェック

      • 特異値の大きい部分が似ていれば、同じ画像もしくは似てる画像なのではないか?

動画 1-5~10: 確率

  • 5: 二種類の確率

    • 両者の定義は異なるが、その後に適用できる数学は共通

    1. 頻度確率 (客観確率) = 発生する頻度

    2. ベイズ確率 (主観確率) = 信念の度合い

  • 6: (頻度)確率の定義

    • P(A) = 事象Aが起こる数 / すべての事象の数 :=n(A)n(U) := \frac{n(A)}{n(U)}:=n(U)n(A)​

      • 0から1の間の値

      • つまり、P(Aˉ)=1−P(A) P(\bar{A}) = 1-P(A)P(Aˉ)=1−P(A)

  • 7: もうちょっと複雑な場合

    • P(A∩B)=P(A)P(B∣A)=P(B)P(A∣B) P(A\cap B) = P(A) P(B|A) = P(B) P(A|B) P(A∩B)=P(A)P(B∣A)=P(B)P(A∣B)

    • P(B∣A) P(B|A)P(B∣A) : Aが起きた条件の元でBが起きる確率

  • 8: 条件付き確率の定義

    • P(A∣B)=P(A∩B)P(B)=n(A∩B)n(B) P(A|B) = \frac{P(A\cap B)}{P(B)} = \frac{n(A\cap B)}{n(B)}P(A∣B)=P(B)P(A∩B)​=n(B)n(A∩B)​

      • 例: 雨が降っている条件下での交通事故に遭う確率

    • 通常 確率計算の全体集合は Uだが、 これを事象Bに限定する

      • これはベン図で書くと明らか

  • 9: 同時確率

    • P(A∩B)=P(A)P(B∣A) P(A\cap B) = P(A)P(B|A)P(A∩B)=P(A)P(B∣A) だが、

    • 事象A,Bが独立なら P(A∩B)=P(A)P(B) P(A\cap B) = P(A) P(B) P(A∩B)=P(A)P(B)

      • P(B∣A)=P(B)P(B|A) = P(B) P(B∣A)=P(B) だから

  • 10: 和集合

    • P(A∪B)=P(A)+P(B)−P(A∩B) P(A\cup B) = P(A) + P(B) - P(A\cap B)P(A∪B)=P(A)+P(B)−P(A∩B)

    • 引き算は、二重に数えている事象の数の分

  • 11: ベイズ則

    • 条件付き確率と、同時確率の式を組合せて導かれる

      • P(B∣A)=P(A∩B)P(A)=P(A∣B)P(B)P(A) P(B|A) = \frac{P(A \cap B )}{P(A)} = \frac{P(A|B)P(B) }{P(A)} P(B∣A)=P(A)P(A∩B)​=P(A)P(A∣B)P(B)​

    • 使用例

      • A: 飴玉持ってる　P(A) = 1/4

      • B: 子供が笑顔 P(B) = 1/3

      • P(B|A) = 1/2

      • 笑顔の子供が飴玉を持っている確率は?

        • P(A∣B)=P(A∩B)/P(B)=P(A)P(B∣A)/P(B) P(A|B) = P(A\cap B) /P(B) = P(A) P(B|A) /P(B) P(A∣B)=P(A∩B)/P(B)=P(A)P(B∣A)/P(B)

        • = 1/4 * 1/2 / (1/3) = 3/8

      • 動画では、"W(x) W(x) W(x) は、x という事象に対する場合の数と捉えると、逆数は確率" というような説明がされていたが、この議論あまり直感的に感じられないなぁ。

        • 前節の例では、"点の数"という整数値で議論されていたが、実際には微分値として扱っているので、"1個の差を見つたい" と言う例にこだわらず, 最初から 場合の数が n 個のときの情報量を f(n)f(n)f(n) で定義できるとして、その関数 fff の形を見つける問題を考えていた、と思った方がわかりやすい.

  • 14 : シャノンエントロピー

    • 自己情報量の期待値 : H(x)=E(I(x))=−E(log(P(x)))=−∑(P(x)log(P(x))) H(x) = E(I(x)) = -E (log(P(x))) = - \sum (P(x) log(P(x))) H(x)=E(I(x))=−E(log(P(x)))=−∑(P(x)log(P(x)))

      • 積分で計算することももちろんあるが、ここでは和文で表記する

    • 例: コイン投げ (表が出る確率 ppp )

  • 15 : カルバック・ライブラー ダイバージェンス (KLダイバージェンス)

    • DKL(P∥Q)=Ex∼P[log⁡P(x)Q(x)]=∑xP(x)log⁡P(x)Q(x) D_{\mathrm{KL}} (P \| Q) = E_{x \sim P } \bigl[ \log \frac{P(x)}{Q(x)} \bigr] = \sum_x P(x) \log \frac{P(x)}{Q(x)}DKL​(P∥Q)=Ex∼P​[logQ(x)P(x)​]=∑x​P(x)logQ(x)P(x)​

      • 期待値の中身 log⁡P(x)Q(x)=I(Q(x))−I(P(x)) \log \frac{P(x)}{Q(x)} = I(Q(x)) - I(P(x)) logQ(x)P(x)​=I(Q(x))−I(P(x)) は 分布 P,QP,QP,Q の情報量の差

    • "距離っぽい" 特性として Q=PQ=PQ=P ならばゼロとなるような量として定義

      • 数学的な意味で距離の公理を満たすわけではないので注意が必要だが、カルバック・ライブラー情報量 とか、かるバック・ライブラー距離 という呼び方をされる

  • 16 : 交差エントロピー

    • H(P,Q)=−Ex∼Plog⁡Q(x)=−∑xP(x)log⁡Q(x) H(P,Q) = -E_{x \sim P} \log Q(x) = -\sum_x P(x) \log Q(x) H(P,Q)=−Ex∼P​logQ(x)=−∑x​P(x)logQ(x)

      • 期待値の中身は、分布Qの自己情報量

      • KLダイバージェンスのような距離っぽさっは無いが、分布 P,QP,QP,Q の関係を表す項だけをシンプルに含む

    • H(P,Q)=H(P)+DKL(P∥Q) H(P,Q) = H(P) + D_{\mathrm{KL}} (P \| Q) H(P,Q)=H(P)+DKL​(P∥Q) の関係

    • しかし、交差エントロピーが使われるようになった経緯は、もともとはKLダーヴァージェンスとは関係ない

      • シャノンエントロピーのような情報の価値を計算したい

      • リアルな状況では、想定した分布 QQQ と観測から得られる分布 PPP はことなる状況が多々ある
